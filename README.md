# 🌻 E-commerce Livraria Girassol 🌻

Este repositório contém o código do projeto final desenvolvido no primeiro bloco do Bootcamp Full Stack de JavaScript da Generation Brasil. O projeto é um sistema de e-commerce para uma livraria, permitindo a realização de operações CRUD (Criar, Ler, Atualizar e Deletar) para gestão de livros.

## 📚 Funcionalidades

- Listagem de livros: Exibe uma lista de todos os livros cadastrados.

- Listagem de livros por ID: Utiliza o ID para listar as informações sobre o livro buscado.

- Cadastro de livros: Permite adicionar novos livros ao sistema.

- Atualização de livros: Possibilita a edição das informações de um livro.

- Exclusão de livros: Remove livros do sistema.

## 📋 Pré-requisitos

- Typescript
- Node.js 

## ⚙️ Instalação

1. Clone o repositório para sua máquina local:

    ```bash
    git clone https://github.com/liamatsubara/projeto_final_bloco_01.git
    ```

2. Instale as dependências:

    ```bash
    npm install
    ```

3. Rode o projeto:

    ```bash
    ts-node Menu.ts
    ```
